## Business Domains

- Auth
- Users
- Departments
- Languages
- Skills
- CVs
- Projects

`app` and `shared` contain "segments" directly. Everything else i.e. `features`, `pages` and `widgets` etc.
contain domain subdirs which in turn contain "segments" like `ui`, `services`, `http` etc.
